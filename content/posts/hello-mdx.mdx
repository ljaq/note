这是一篇示例文章。正文支持 **粗体**、列表与 `行内代码`。

## 小节标题

- 文章放在 `content/posts/*.mdx`
- 构建时通过 `generateStaticParams` 全量预渲染
- 导航与按钮在 `components/ui/` 中复用

```ts
export async function generateStaticParams() {
  return getPostSlugs().map(slug => ({ slug }))
}
```

更多内容直接编辑此文件即可。
